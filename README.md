# REAL NVP and EBM

This project implements a combination of Real NVP (Normalizing Flows) and Energy-Based Models (EBM) for generative modeling on the MNIST dataset.

## Overview

The code trains two generative models:
- **Real NVP**: A normalizing flow model that learns a reversible transformation from data to a latent space
- **EBM**: An energy-based model that learns to assign low energy to real data and high energy to fake data

Both models are trained jointly on MNIST digits and can generate new samples.

## Dependencies

- PyTorch
- Torchvision
- Matplotlib
- Tqdm

## Installation

Install the required packages:

```bash
pip install torch torchvision matplotlib tqdm
```

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/VIGNESH4777/REAL_NVP-AND-EBM-.git
cd REAL_NVP-AND-EBM-
```

2. Run the training script:
```bash
python main.py
```

The script will:
- Download the MNIST dataset automatically
- Train both Real NVP and EBM models for 2 epochs (demo)
- Generate and display sample images from both models

## Output

After training, the script will display a visualization showing:
- Top row: Samples generated by Real NVP
- Bottom row: Samples generated by EBM

## Architecture Details

- **Real NVP**: Uses 6 coupling layers with alternating checkerboard masks
- **EBM**: Uses a simple MLP with LeakyReLU activations
- **Training**: Alternates between maximum likelihood for NVP and contrastive divergence for EBM

## Note

This is a demo implementation running for only 2 epochs. For better results, increase the number of epochs in the training loop.
